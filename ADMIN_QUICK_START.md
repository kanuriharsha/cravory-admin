# 🎯 Quick Start: Adding Restaurant Location

> **New Feature:** You can now add restaurant locations using Google Maps links!

## 📍 Two Easy Ways to Add Location

### Option 1: Google Maps Link (Recommended)

**1. Find the restaurant on Google Maps**
   - Go to [Google Maps](https://maps.google.com)
   - Search for the restaurant

**2. Copy the link**
   - Click the "Share" button
   - Click "Copy link"
   - (Or copy URL from browser address bar)

**3. Paste in admin panel**
   - Open "Add Restaurant" dialog
   - Find "Google Maps Link" field
   - Paste the link
   - ✅ Coordinates extracted automatically!

**Example:**
```
Link: https://www.google.com/maps/@17.4435,78.3772,15z
→ Latitude: 17.4435
→ Longitude: 78.3772
```

---

### Option 2: Manual Entry

**1. Right-click on Google Maps**
   - Find the exact location
   - Right-click on the spot
   - Click the coordinates that appear

**2. Enter in admin panel**
   - Latitude field: First number (e.g., 17.4435)
   - Longitude field: Second number (e.g., 78.3772)

---

## ✨ What Happens Next?

Once coordinates are saved:
- 🗺️ System can calculate distance from customer to restaurant
- 📍 Accurate delivery radius checks
- 🎯 Sort restaurants by proximity to customer
- 📊 Better delivery partner assignment

## 🚨 Required Field

**Location is now mandatory** - you must provide either:
- ✅ Google Maps link (auto-extracts coordinates), OR
- ✅ Latitude + Longitude manually

## 💡 Tips

1. **Always verify coordinates appear**
   - After pasting Maps link, check lat/lng fields
   - Should see success message ✅

2. **Hyderabad coordinates typically:**
   - Latitude: 17.x (between 17.0 - 18.0)
   - Longitude: 78.x (between 78.0 - 79.0)

3. **If Maps link doesn't work:**
   - Try copying from browser address bar instead
   - Or enter coordinates manually

4. **Common Hyderabad areas:**
   - HITEC City: 17.4435, 78.3772
   - Gachibowli: 17.4400, 78.3489
   - Banjara Hills: 17.4126, 78.4401

## ❓ FAQ

**Q: Where do I find the Google Maps link?**
A: Search for restaurant on Google Maps → Click "Share" → "Copy link"

**Q: What if I get an error?**
A: Enter latitude and longitude manually using right-click method

**Q: Are short links (goo.gl) supported?**
A: Sometimes - but full URLs work better. If short link fails, open it in browser and copy the full URL.

**Q: Can I update location later?**
A: Yes! Edit restaurant → Update Maps link or coordinates → Save

**Q: What if restaurant has multiple locations?**
A: Add each location as a separate restaurant entry with its own coordinates
